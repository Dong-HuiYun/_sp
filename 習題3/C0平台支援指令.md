# C0 平台支援的組合語言指令集

## 支援的指令

### **1. LDI - 載入立即值**

```assembly

LDI R0 5      ; R0 = 5
LDI R1 100    ; R1 = 100
LDI R7 0      ; R7 = 0

```

**功能**：將一個常數值放入指定的暫存器

---

### **2. ADD - 暫存器相加**

```assembly

LDI R1 10
LDI R2 20
ADD R3 R1     ; R3 = R1 + R2? 不對，正確寫法：
ADD R3 R1     ; R3 = R3 + R1 (累加到自己)

```
> ⚠️ **注意**：ADD 的語法是 `ADD 目標 來源`，會將**目標 + 來源**的結果存回**目標**

**正確用法**：

```assembly

LDI R1 10
LDI R2 20
ADD R1 R2     ; R1 = 10 + 20 = 30

```

---

### **3. SUBI - 減去立即值**

```assembly

LDI R1 100
SUBI R1 1     ; R1 = 100 - 1 = 99
SUBI R2 5     ; R2 = 0 - 5 = -5 (支援負數)

```

**功能**：將暫存器減去一個常數值

---

### **4. BNZ - 非零則跳轉**

```assembly

loop:
    SUBI R1 1
    BNZ R1 loop   ; 如果 R1 ≠ 0，跳回 loop 標籤

```

**功能**：檢查暫存器，如果不是零就跳轉到指定標籤

---

### **5. HALT - 停止程式**

```assembly

HALT          ; 結束執行

```

**功能**：終止程式執行，連續執行模式會在此停止

---

## 完整測試程式範例

### **範例 1：簡單加法**
```assembly
LDI R1 5
LDI R2 3
ADD R1 R2
HALT
```
**結果**：R1 = 8, R2 = 3

---

### **範例 2：累加 1~N（內建範例）**

```assembly

LDI R1 10     ; 設定 N=10
LDI R2 0      ; 總和初始值
loop:
ADD R2 R1     ; 累加當前數字
SUBI R1 1     ; 數字減 1
BNZ R1 loop   ; 還沒到 0 就繼續
HALT

```
**結果**：R2 = 55（1+2+...+10）

---

### **範例 3：條件判斷**

```assembly

LDI R1 0
LDI R2 100
BNZ R1 skip   ; R1=0，不跳轉
ADD R2 R2     ; 會執行這行
skip:
HALT

```
