# 繁體中文程式語言

一個使用**全繁體中文關鍵字**的程式語言，包含完整的 Lexer、Parser、Compiler 與 Stack-based VM。

---

## 檔案結構

```
lexer_parser.py   — Lexer（詞法分析）+ Parser（語法分析）+ AST 定義
compiler_vm.py    — Compiler（編譯器）+ VM（堆疊虛擬機）+ REPL + CLI
範例.中文          — 範例程式碼
```

## 安裝需求

只需要 **Python 3.8+**，無需安裝任何套件。

---

## 執行方式

### 1. 執行示範程式（最快）

```bash
python compiler_vm.py
```

輸出 5 個內建示範：算術、字串、條件判斷、循環、FizzBuzz。

---

### 2. 執行 `.中文` 原始碼檔案

```bash
python compiler_vm.py 範例.中文
```

加上 `--bytecode` 可以印出 Bytecode 反組譯：

```bash
python compiler_vm.py 範例.中文 --bytecode
```

加上 `--debug` 可以印出 VM 逐步執行追蹤（每條指令 + 堆疊狀態）：

```bash
python compiler_vm.py 範例.中文 --debug
```

加上 `--ast` 可以印出抽象語法樹：

```bash
python compiler_vm.py 範例.中文 --ast
```

---

### 3. 互動式 REPL

```bash
python compiler_vm.py repl
```

---

### 4. 在 Python 中直接呼叫

```python
from compiler_vm import run_source

source = """
令 名字 為 字串 等於 「世界」。
顯示 「你好，」 加 名字。
"""

run_source(source)
# 輸出：你好，世界

# 顯示 Bytecode 反組譯 + AST + VM 追蹤
run_source(source, show_bytecode=True, show_ast=True, debug=True)
```

## 語法速查

### 變數宣告
```
令 名稱 為 整數 等於 100。
令 文字 為 字串 等於 「你好」。
```

### 賦值
```
令 名稱 等於 200。
```

### 輸出
```
顯示 名稱。
顯示 「直接顯示字串」。
```

### 算術運算
```
令 結果 為 整數 等於 甲 加 乙。   # +
令 結果 為 整數 等於 甲 減 乙。   # -
令 結果 為 整數 等於 甲 乘 乙。   # *
令 結果 為 整數 等於 甲 除 乙。   # / (整數除法)
```

### 字串串接
```
令 全句 為 字串 等於 「前面」 加 「後面」。
```

### 條件判斷
```
如果（條件）則：
  語句。
。完否則：
  語句。
。完
```

### 循環
```
當（條件）就：
  語句。
。完
```

## 型態系統

- **強型態**：宣告時必須指定 `整數` 或 `字串`
- **型態檢查**：在編譯期（Compiler）與執行期（VM）都會檢查
- `整數` 加 `整數` → 整數加法
- `字串` 加 `字串` → 字串串接
- 不同型態混用會報錯

---

## 架構說明

```
原始碼字串
   ↓
 Lexer（詞法分析器）
   → Token 串列
   ↓
 Parser（語法分析器）
   → AST（抽象語法樹）
   ↓
 Compiler（編譯器）
   → Bytecode 指令列表
   ↓
 VM（堆疊虛擬機）
   → 程式執行輸出
```

### Bytecode 指令集

| 指令             | 說明                          |
|------------------|-------------------------------|
| PUSH_INT \<n\>   | 將整數常數推入堆疊             |
| PUSH_STR \<s\>   | 將字串常數推入堆疊             |
| LOAD \<name\>    | 將變數值推入堆疊               |
| STORE \<name\>   | 彈出頂端值並存入變數           |
| DECLARE \<n,t\>  | 宣告新變數（型態記錄）         |
| ADD / SUB / MUL / DIV | 四則運算               |
| NEG              | 取負值                        |
| EQ / NEQ         | 等於 / 不等於比較              |
| GT / LT / GTE / LTE | 大小比較                  |
| AND / OR / NOT   | 邏輯運算                      |
| JUMP \<addr\>    | 無條件跳轉                    |
| JUMP_IF_FALSE \<addr\> | 條件跳轉（假則跳）      |
| PRINT            | 彈出頂端值並印出               |
| HALT             | 程式結束                      |
