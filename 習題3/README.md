# C0 Computer Visualizer

一個基於網頁的輕量級 **C0 電腦架構教學與視覺化平台**。

這個專案旨在將抽象的 CPU 指令執行過程（暫存器變化、程式計數器跳轉、記憶體讀取）視覺化，幫助初學者理解電腦底層是如何運作的。

## 特色功能

-   **即時編譯 (Immediate Assembly)：** 隨寫隨跑，將組合語言轉換為機器邏輯。
-   **單步執行 (Step Execution)：** 支援逐行執行指令，觀察 CPU 在每一個時脈週期的變化。
-   **暫存器監控：** 即時顯示 `PC` (程式計數器) 與 `R0-R7` 通用暫存器的數值。
-   **視覺化記憶體：** 透過高亮效果顯示 CPU 目前正在讀取哪一個記憶體區塊。
-   **輕量化設計：** 僅需一個 `index.html` 檔案，無須安裝複雜環境。

## 指令集說明 (ISA)

本模擬器支援以下 C0 指令：

| 指令 | 格式 | 說明 |
| :--- | :--- | :--- |
| **LDI** | `LDI Rd Imm` | **Load Immediate**: 將常數 `Imm` 存入暫存器 `Rd`。 |
| **ADD** | `ADD Rd Rs` | **Add**: 將 `Rd` 與 `Rs` 的值相加，結果存回 `Rd`。 |
| **SUBI** | `SUBI Rd Imm` | **Subtract Immediate**: 將 `Rd` 減去常數 `Imm`。 |
| **BNZ** | `BNZ Rd Label` | **Branch if Not Zero**: 若 `Rd` 不為 0，則跳轉至 `Label` 處。 |
| **HALT** | `HALT` | **Halt**: 停止程式執行。 |

## 快速上手指南

1.  **編寫代碼：** 在左側編輯器輸入指令。預設程式碼為計算 1 加到 10 的總和。
2.  **組譯 (Assemble)：** 點擊 `1. Assemble` 按鈕。系統會掃描標籤 (Label) 並初始化 CPU 狀態。
3.  **執行 (Step)：** 連續點擊 `2. Step Execution`。
    *   觀察 **PC** 如何遞增。
    *   觀察 **R1** 如何進行倒數 (10, 9, 8...)。
    *   觀察 **R2** 如何累加總和。
4.  **迴圈觀察：** 當執行到 `BNZ` 時，若條件成立，你會看到 **PC** 跳回上方，這就是電腦「迴圈」的真相。

## 範例程式碼：1 加到 10

```asm
LDI R1 10    ; R1 作為計數器，從 10 開始
LDI R2 0     ; R2 作為累加器，初值為 0
loop:        ; 迴圈標籤
ADD R2 R1    ; R2 = R2 + R1
SUBI R1 1    ; R1 = R1 - 1
BNZ R1 loop  ; 如果 R1 還不是 0，跳回 loop
HALT         ; 結束，最終 R2 會顯示 55
```

## 教育意義

本專案模擬了 **馮·紐曼架構 (Von Neumann Architecture)** 的核心循環：
1.  **Fetch (擷取)：** 從記憶體讀取指令（由黃色高亮表示）。
2.  **Decode (解碼)：** 判斷指令類型（LDI, ADD 等）。
3.  **Execute (執行)：** 更新暫存器數值或改變程式流向。
